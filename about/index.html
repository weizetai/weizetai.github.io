负责工作：

  1. 采用基于 webpack 的 Vue 进行项目构建，使用 git 进行代码版本管理
  2. 使用 Vue-router 实现页面跳转

项目 2 国家自然科学基金面上项目 混行交通中时变非高斯噪声和定位失效下智能网联汽车目标跟踪

项目描述：针对真实混行交通环境中时变非高斯噪声和定位失效情况，提出基于一致性点漂移的多目标跟 踪方法，以期提高智能网联汽车的目标跟踪精度和鲁棒性

负责工作：

  1. 搭建智能车环境感知平台，负责激光雷达和相机标定，利用 ROS 系统时间戳实现时间同步
  2. 在采集数据的基础上，使用 python/MATLAB 编程实现所提算法进行实地校园环境数据验证

项目 3 重庆市自然科学基金面上项目 山地环境下基于变分贝叶斯的智能网联汽车目标协同检测研究

项目描述：为了实现山地复杂多变环境下智能网联汽车的目标检测，本项目主要涉及面向智能网联汽车的 多异质车载传感器目标检测模型领域，开发基于 ROS 系统下的目标检测平台

负责工作：

  1. 负责基于图像和激光雷达的环境感知，实现基于图像和激光雷达的目标检测
  2. 实现图像和激光雷达信息融合的目标检测

成 果：Vehicle Detection Based on Lidar-Camera Fusion Using Wasserstein Distance Method（EI 会议在审）
